## 3. Goals

### Primary Goals

- Reduce manual prompt copying between AI agents.
    
- Centralize all AI collaboration.
    
- Make AI software development transparent.
    
- Allow Codex to orchestrate an entire development team.
    

### Secondary Goals

- Project organization.
    
- Task tracking.
    
- Code review.
    
- Agent monitoring.
    

---

## 4. User Roles

### Developer (User)

Creates projects, gives high-level instructions, reviews progress, and approves major decisions.

### Codex

Lead engineer and project manager.

Responsibilities:

- Understand project goals.
    
- Break work into subtasks.
    
- Complete the hardest work itself.
    
- Delegate suitable tasks.
    
- Review all completed work.
    
- Maintain project consistency.
    

### Worker Agents

Examples:

- AntiGravity
    
- OpenCode
    
- Blackbox
    
- Kilocode
    
- Mimo code
    

Responsibilities:

- Execute assigned work.
    
- Report progress.
    
- Submit completed work.
    
- Never self-assign tasks.
